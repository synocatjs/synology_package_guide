# Provilege 文件

`privilege` 文件定义了包的**权限和运行环境**。

```json
{
    "defaults": {
        "run-as": "package"
    }
}
```

## 字段说明

| 字段 | 说明 | 可选值 |
|------|------|--------|
| `run-as` | 指定包运行时使用的用户身份 | `"package"`、`"root"`、`"nobody"` |

## 运行用户选项

| 值 | 说明 | 使用场景 |
|---|------|----------|
| `"package"` | 以包专用用户运行（推荐） | 大多数应用，安全隔离 |
| `"root"` | 以 root 身份运行 | 需要系统级权限的应用 |
| `"nobody"` | 以 nobody 用户运行 | 最低权限，只读操作 |
